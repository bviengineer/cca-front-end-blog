What is HTML?

HTML is an acronym for Hpertext Markup Language. Hypertext is the ability to jump from one resource to another, one website to another or from a particular location on a website to a different location on the same website.

Hypertext is the foundation of the web as it connects various resoruces together much like a spider web, hence the name web.

Markup in the context of the web means, adding special notes to a document via HTML tags, that describes its' meaning and defines its' structure.

Below is an example of a hyperlink and the website that it redirects the user to. The area underlined in red consist of the HTML tag, its attributes, the URL or web address of the resource you will be redirected to after clicking on the link as well as the name the link will display as.

There have been many versions of HTML since its' inception and the most recent version, HTML 5, added new tags, most notably, tags that provide better semantic meaning.

You can learn more about the history of HTML on Wikipedia.
